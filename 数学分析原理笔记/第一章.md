# 第一章 - 实数与复数系统

## 1.1 示例
**原文**：证明方程 p² = 2 在有理数范围内无解。

**证明**：[理解性翻译]：
假设存在有理数 p = m/n（最简形式）满足 p² = 2。则 m² = 2n²，故 m² 为偶数，m 也为偶数。设 m = 2k，代入得 4k² = 2n²，即 n² = 2k²，故 n 也为偶数。这与 m/n 为最简形式矛盾。

## 1.2 备注
**原文**：上述讨论的目的是表明有理数系统存在某些"空隙"，尽管在任何两个有理数之间都存在另一个有理数。序列 1.4, 1.41, 1.414, 1.4142,... 趋向于 √2，但 √2 不是有理数。

## 1.3 定义
**原文**：如果 A 是任意集合（其元素可以是数或其他对象），我们记 x ∈ A 表示 x 是 A 的成员（或元素）。如果 x 不是 A 的成员，我们记 x ∉ A。

## 1.4 定义
**原文**：在本章中，所有有理数的集合记为 Q。

## 1.5 定义（序）
**原文**：设 S 是一个集合。S 上的序是一个关系，记作 <，满足以下两个性质：
1. 如果 x,y ∈ S，则 x<y, x=y, y<x 中恰好有一个成立。
2. 如果 x,y,z ∈ S，且 x<y 和 y<z，则 x<z。

## 1.6 定义（有序集）
**原文**：有序集是在其上定义了序的集合 S。

## 1.7 定义（上界）
**原文**：设 S 是有序集，E ⊆ S。如果存在 β ∈ S 使得对每个 x ∈ E 有 x ≤ β，则称 E 有上界，并称 β 是 E 的一个上界。

## 1.8 定义（最小上界）
**原文**：设 S 是有序集，E ⊆ S，且 E 有上界。如果存在 α ∈ S 满足以下性质：
1. α 是 E 的上界。
2. 如果 γ < α，则 γ 不是 E 的上界。
那么 α 称为 E 的最小上界或上确界，记作 α = sup E。

## 1.9 例子
**原文**：考虑例 1.1 中的集合 A 和 B 作为有序集 Q 的子集。集合 A 有上界。事实上，A 的上界恰好是 B 的成员。集合 B 有下界；B 的下界是 A 的成员。

## 1.10 定义（最小上界性质）
**原文**：如果以下条件成立，则称有序集 S 具有最小上界性质：如果 E ⊆ S，E 非空，且 E 有上界，则 sup E 存在于 S 中。

## 1.11 定理
**原文**：设 S 是具有最小上界性质的有序集，B ⊆ S，B 非空且有下界。令 L 是 B 的所有下界的集合。则 α = sup L 存在于 S 中，且 α = inf B。

**证明**：[理解性翻译]：由于 B 有下界，L 非空。由最小上界性质，α = sup L 存在。验证 α 是 B 的最大下界。

## 1.12 定义（域）
**原文**：域是一个集合 F，带有两种运算，称为加法和乘法，满足域公理（加法的 A1-A4，乘法的 M1-M4，以及分配律 D）。

## 1.13 备注
**原文**：讨论域的基本性质，如零元、负元、单位元、逆元等。

## 1.14 命题（加法性质）
**原文**：加法公理蕴含以下陈述：
1. 如果 x+y = x+z，则 y=z。
2. 如果 x+y = x，则 y=0。
3. 如果 x+y = 0，则 y=-x。
4. -(-x) = x。

## 1.15 命题（乘法性质）
**原文**：乘法公理蕴含类似的陈述。

## 1.16 命题（域性质）
**原文**：域公理蕴含以下陈述，对任意 x,y,z ∈ F：
1. 0x = 0。
2. (-x)y = -(xy) = x(-y)。
3. (-x)(-y) = xy。

## 1.17 定义（有序域）
**原文**：有序域是一个域 F，同时也是一个有序集，满足：
1. 如果 y<z，则 x+y < x+z。
2. 如果 x>0 且 y>0，则 xy > 0。

## 1.18 命题（有序域性质）
**原文**：在每个有序域中以下陈述成立：
1. x>0 ⇒ -x<0。
2. x>0 且 y<z ⇒ xy<xz。
3. x<0 且 y<z ⇒ xy>xz。
4. x≠0 ⇒ x²>0。
5. 0<1。

## 1.19 定理（实数存在性）
**原文**：存在一个具有最小上界性质的有序域 R。R 的成员称为实数。

**证明**：[理解性翻译]：通过戴德金分割构造实数，验证其满足所有域公理和序性质，并具有最小上界性质。

## 1.20 定理（阿基米德性质）
**原文**：如果 x,y ∈ R 且 x>0，则存在正整数 n 使得 nx>y。

## 1.21 定理（n 次方根存在性）
**原文**：对每个实数 x>0 和每个整数 n>0，存在唯一正实数 y 使得 yⁿ = x。

**证明**：[理解性翻译]：构造集合 E = {t ∈ R : tⁿ < x}，利用最小上界性质证明 y = sup E 满足 yⁿ = x。

## 1.22 小数
**原文**：讨论实数与十进制小数的关系。

## 1.23 定义（扩展实数系）
**原文**：扩展实数系由实数域 R 和两个符号 +∞ 和 -∞ 组成。我们保持 R 中的原始序，并对每个 x ∈ R 定义 -∞ < x < +∞。

## 1.24 定义（复数）
**原文**：复数是有序对 (a,b)，其中 a,b ∈ R。"有序"意味着如果 a≠b，则 (a,b) 和 (b,a) 被视为不同的。

## 1.25 定理（复数域）
**原文**：这些加法和乘法的定义使得所有复数的集合成为一个域，其中 (0,0) 和 (1,0) 分别扮演 0 和 1 的角色。

## 1.26 定理（实数嵌入）
**原文**：对任意实数 a 和 b，有 (a,0)+(b,0) = (a+b,0) 和 (a,0)(b,0) = (ab,0)。

## 1.27 定义（虚数单位）
**原文**：i = (0,1)。

## 1.28 定理
**原文**：i² = -1。

## 1.29 定理
**原文**：如果 a 和 b 是实数，则 (a,b) = a+bi。

## 1.30 定义（共轭）
**原文**：如果 a,b 是实数且 z = a+bi，则复数 z̄ = a-bi 称为 z 的共轭。数 a 和 b 分别称为 z 的实部和虚部。

## 1.31 定理（共轭性质）
**原文**：如果 z 和 w 是复数，则：
1. z+w̄ = z̄+w̄
2. zw̄ = z̄·w̄
3. z+z̄ = 2Re(z)
4. z-z̄ = 2iIm(z)
5. zz̄ ≥ 0，等号仅当 z=0 时成立

## 1.32 定义（绝对值）
**原文**：如果 z 是复数，其绝对值 |z| 是 zz̄ 的非负平方根；即 |z| = (zz̄)^{1/2}。

## 1.33 定理（绝对值性质）
**原文**：设 z 和 w 是复数，则：
1. |z| ≥ 0，等号仅当 z=0 时成立
2. |z̄| = |z|
3. |zw| = |z||w|
4. |Re z| ≤ |z|
5. |z+w| ≤ |z|+|w|（三角不等式）

## 1.34 记号
**原文**：如果 x₁,...,xₙ 是复数，我们记 x₁+...+xₙ = ∑_{j=1}^{n} x_j 和 x₁·...·xₙ = ∏_{j=1}^{n} x_j。

## 1.35 定理（柯西-施瓦茨不等式）
**原文**：如果 a₁,...,aₙ 和 b₁,...,bₙ 是复数，则 |∑a_jb_j|² ≤ (∑|a_j|²)(∑|b_j|²)。

## 1.36 定义（欧几里得空间）
**原文**：对每个正整数 k，令 Rᵏ 是所有有序 k 元组 x=(x₁,...,xₖ) 的集合，其中 x₁,...,xₖ ∈ R。

## 1.37 定理（向量运算）
**原文**：设 x,y,z ∈ Rᵏ，且 α 是实数，则：
1. x+y = y+x
2. (x+y)+z = x+(y+z)
3. x+0 = x
4. x+(-x) = 0
5. α(x+y) = αx+αy
6. (α+β)x = αx+βx
7. (αβ)x = α(βx)
8. 1x = x

## 1.38 备注
**原文**：定理 1.37(a),(b),(f) 将允许我们将 Rᵏ 视为度量空间。R¹ 通常称为直线或实直线。同样，R² 称为平面或复平面。
