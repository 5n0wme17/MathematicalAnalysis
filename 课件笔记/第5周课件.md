# 第5周课件 - 级数收敛性与度量空间极限

## 单词表

| 中文术语 | 英文术语 | 定义/说明 |
|---------|---------|---------|
| 级数收敛 | Series convergence | 部分和序列Sₙ = a₁ + ... + aₙ收敛 |
| 绝对收敛 | Absolute convergence | ∑|aₙ|收敛 ⇒ ∑aₙ收敛 |
| 条件收敛 | Conditional convergence | ∑aₙ收敛但∑|aₙ|发散 |
| 一致收敛 | Uniform convergence | ∀ε>0, ∃N, ∀n≥N, ∀x∈X, |fₙ(x)-f(x)|<ε |
| 幂级数 | Power series | 形如∑cₙ(x-a)ⁿ的级数 |
| 收敛半径 | Radius of convergence | R = 1/limsup |cₙ|^(1/n) |

## 课件概述
本课件主要讨论级数的收敛性，特别是条件收敛、绝对收敛，以及度量空间中映射的极限概念。内容涉及Abel-Dirichlet判别法、幂级数收敛性等。

## 主要内容

### 1. 条件收敛与绝对收敛
#### 定义
- **绝对收敛**：∑|aₙ|收敛 ⇒ ∑aₙ收敛
- **条件收敛**：∑aₙ收敛但∑|aₙ|发散

**课件原证明语句**：
"设Sₙ = Σ|aₖ|收敛，由柯西准则，∀ε>0, ∃N, ∀n>m≥N, Σ_{k=m+1}ⁿ|aₖ|<ε，故|Σaₖ|≤Σ|aₖ|<ε"

**思考问题**：
1. 为什么绝对收敛级数可以重排而不改变和？
2. 构造一个条件收敛级数，使其重排后发散

#### 例子
课件中提到了一个条件收敛的例子，但具体细节不完整。常见的条件收敛例子包括：
- 交错调和级数：∑(-1)ⁿ⁺¹/n
- 其他交错级数满足Leibniz判别法条件

**课件原证明语句**：
"交错调和级数∑(-1)ⁿ⁺¹/n条件收敛：由Leibniz判别法知其收敛，但∑1/n发散"

**思考问题**：
如何证明交错调和级数条件收敛？

### 2. Abel-Dirichlet判别法
#### 定理（Abel判别法）
设∑aₙ收敛，{bₙ}单调有界，则∑aₙbₙ收敛。

**课件原证明语句**：
"利用Abel变换：设Aₙ = Σ_{k=1}ⁿaₖ，则Σaₖbₖ = Aₙbₙ - ΣAₖ(b_{k+1}-bₖ)，由单调有界性得证"

**思考问题**：
1. 如果{bₙ}单调但无界，结论是否成立？
2. 举例说明Abel判别法的应用

#### 定理（Dirichlet判别法）
设∑aₙ的部分和有界，{bₙ}单调趋于0，则∑aₙbₙ收敛。

**课件原证明语句**：
"设Sₙ = Σ_{k=1}ⁿaₖ有界，由Abel变换得Σaₖbₖ = Sₙbₙ - ΣSₖ(b_{k+1}-bₖ)，利用单调趋于0得证"

**思考问题**：
比较Abel判别法和Dirichlet判别法的异同

### 3. 幂级数收敛性
#### 定理（Abel定理）
设c₀ ≥ c₁ ≥ c₂ ≥ ... ≥ 0，且lim cₙ = 0，则幂级数∑cₙzⁿ在|z| ≤ 1上收敛（可能在z=1处除外）。

**课件原证明语句**：
"当|z|=1且z≠1时，部分和|Σ_{k=0}ⁿzᵏ| = |(1-zⁿ⁺¹)/(1-z)| ≤ 2/|1-z|有界，由Dirichlet判别法得证"

**证明思路**：
- 利用Abel变换或Dirichlet判别法
- 当|z| = 1但z ≠ 1时，部分和∑zⁿ有界
- z=1时退化为∑cₙ，需要额外条件

**思考问题**：
1. 为什么在z=1处需要额外条件？
2. 举例说明Abel定理的应用

#### 收敛半径
对于一般幂级数∑aₙzⁿ，收敛半径R = 1/limsup |aₙ|^(1/n)

**课件原证明语句**：
"由Cauchy-Hadamard公式：R = 1/limsup |aₙ|^{1/n}，在|z|<R内绝对收敛，|z|>R发散"

**思考问题**：
如何计算具体幂级数的收敛半径？

### 4. 度量空间中的极限
#### 两个等价的极限定义
设f: X → Y为度量空间之间的映射，x₀ ∈ X'（X的极限点），则以下等价：
1. ∀ε > 0, ∃δ > 0, 当0 < d_X(x, x₀) < δ时，有d_Y(f(x), L) < ε
2. 对X中任何收敛到x₀的序列{xₙ}（xₙ ≠ x₀），有f(xₙ) → L

**课件原证明语句**：
"(1)⇒(2)：取δ对应ε，当n充分大时d(xₙ,x₀)<δ，则d(f(xₙ),L)<ε；(2)⇒(1)：反证法，若(1)不成立，构造序列{xₙ}使d(f(xₙ),L)≥ε"

**思考问题**：
1. 为什么需要xₙ ≠ x₀的条件？
2. 举例说明两个定义的等价性

#### 性质
- 极限若存在则唯一
- 极限的代数运算：和、差、积、商（分母极限不为0）
- 复合函数的极限

**课件原证明语句**：
"唯一性证明：假设两个极限L₁≠L₂，取ε=|L₁-L₂|/2导出矛盾"

**思考问题**：
如何证明极限运算的代数性质？

### 5. 级数乘积
#### Cauchy乘积
设∑aₙ和∑bₙ收敛，其Cauchy乘积定义为∑cₙ，其中cₙ = ∑_{k=0}ⁿ aₖb_{n-k}

#### Mertens定理
若∑aₙ绝对收敛，∑bₙ收敛，则其Cauchy乘积收敛且等于(∑aₙ)(∑bₙ)

## 重要定理总结
1. **Abel定理**：单调趋于0的系数幂级数在单位圆盘（除z=1外）收敛
2. **Dirichlet判别法**：部分和有界+单调趋于0 ⇒ 级数收敛
3. **Abel判别法**：收敛级数×单调有界序列 ⇒ 级数收敛
4. **极限的序列刻画**：函数极限 ⇔ 序列极限

## 应用说明
1. 判断交错级数的收敛性
2. 分析幂级数的收敛域
3. 计算级数的乘积
4. 证明函数极限的存在性
