## 课件概述
本课件主要讨论函数序列和函数级数的收敛性，特别是点态收敛与一致收敛的区别，以及一致收敛条件下极限函数的连续性、可积性和可微性。

## 主要内容

### 1. 函数序列的收敛性
#### 点态收敛
设fₙ, f: E → ℝ，称fₙ在E上点态收敛到f，如果对每个x∈E，有lim_{n→∞} fₙ(x) = f(x)。

**课件原证明语句**：
"点态收敛的极限函数唯一：若fₙ→f且fₙ→g点态，则f=g"

#### 一致收敛
称fₙ在E上一致收敛到f，如果：
∀ε>0, ∃N∈ℕ, 使得∀n≥N, ∀x∈E, 有|fₙ(x)-f(x)| < ε

**课件原证明语句**：
"一致收敛⇒点态收敛，但反之不成立（如fₙ(x)=xⁿ在[0,1]上）"

#### 一致收敛的否定形式
fₙ不一致收敛到f ⇔ ∃ε>0, 使得∀N∈ℕ, ∃n≥N, ∃x∈E, 满足|fₙ(x)-f(x)| ≥ ε

**课件原证明语句**：
"用否定形式证明fₙ(x)=xⁿ在[0,1]上不一致收敛到0：取ε=1/2，∀N, ∃n≥N, ∃x=2^{-1/n}, xⁿ=1/2≥ε"

### 2. 一致收敛的柯西准则
fₙ在E上一致收敛 ⇔ ∀ε>0, ∃N∈ℕ, 使得∀m,n≥N, ∀x∈E, 有|fₙ(x)-fₘ(x)| < ε

**课件原证明语句**：
"必要性：|fₙ(x)-fₘ(x)| ≤ |fₙ(x)-f(x)| + |f(x)-fₘ(x)| < ε/2 + ε/2 = ε"

### 3. 一致收敛与连续性
#### 定理
若fₙ在E上连续且一致收敛到f，则f在E上连续。

**课件原证明语句**：
"ε/3论证：|f(x)-f(a)| ≤ |f(x)-fₙ(x)| + |fₙ(x)-fₙ(a)| + |fₙ(a)-f(a)| < ε"

### 4. 一致收敛与积分
#### 定理
设fₙ在[a,b]上黎曼可积且一致收敛到f，则：
1. f在[a,b]上黎曼可积
2. lim_{n→∞} ∫_a^b fₙ(x)dx = ∫_a^b f(x)dx

**课件原证明语句**：
"|∫fₙ-∫f| ≤ ∫|fₙ-f| ≤ (b-a)sup|fₙ-f| → 0"

#### 反例
点态收敛不足以保证积分与极限可交换：
fₙ(x) = n²x(1-x)ⁿ在[0,1]上点态收敛到0，但∫_0^1 fₙ(x)dx → 1 ≠ 0

**课件原证明语句**：
"计算得∫fₙ = n²/(n+1)(n+2) → 1，而∫limfₙ=0"

### 5. 一致收敛与微分
#### 定理
设fₙ在[a,b]上可导，fₙ'在[a,b]上一致收敛到g，且存在x₀∈[a,b]使得{fₙ(x₀)}收敛，则：
1. fₙ在[a,b]上一致收敛到某个f
2. f在[a,b]上可导且f' = g

**课件原证明语句**：
"fₙ(x)=fₙ(x₀)+∫_{x₀}^x fₙ'(t)dt → f(x₀)+∫_{x₀}^x g(t)dt，由微积分基本定理得f'=g"

### 6. 函数级数
#### 定义
∑_{n=1}∞ fₙ(x)的部分和Sₙ(x) = ∑_{k=1}ⁿ fₖ(x)

**课件原证明语句**：
"级数收敛⇔部分和序列收敛，一致收敛⇔部分和序列一致收敛"

#### 一致收敛
∑fₙ在E上一致收敛 ⇔ 部分和序列{Sₙ}在E上一致收敛

**课件原证明语句**：
"柯西准则：∀ε>0, ∃N, ∀m>n≥N, ∀x∈E, |∑_{k=n+1}^m fₖ(x)|<ε"

#### Weierstrass M-判别法
若|fₙ(x)| ≤ Mₙ对一切x∈E成立，且∑Mₙ收敛，则∑fₙ在E上一致收敛。

**课件原证明语句**：
"由∑Mₙ收敛，根据柯西准则，余项∑_{k=n+1}^∞Mₖ→0，且|∑_{k=n+1}^∞fₖ(x)|≤∑_{k=n+1}^∞Mₖ"

### 7. 范数向量空间
#### 定义
设V是函数空间，定义范数‖f‖ = sup_{x∈E} |f(x)|，则(V, ‖·‖)构成赋范向量空间。

**课件原证明语句**：
"一致收敛⇔在sup范数下收敛：‖fₙ-f‖→0 ⇔ fₙ⇉f"

#### 性质
- 一致收敛等价于在sup范数下收敛
- 完备的赋范向量空间称为巴拿赫空间

**课件原证明语句**：
"连续函数空间C[a,b]在sup范数下是巴拿赫空间"

## 重要概念总结
1. **点态收敛**：逐点收敛，不保持函数性质
2. **一致收敛**：整体收敛，保持连续性、可积性、可微性（在适当条件下）
3. **柯西准则**：一致收敛的等价刻画
4. **M-判别法**：判断函数级数一致收敛的实用工具
