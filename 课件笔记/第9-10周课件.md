## 课件概述
本课件主要讨论黎曼积分的定义、可积性条件，以及曲线长度的定义和性质。内容涉及积分的存在性问题、连续函数的可积性等。

## 主要内容

### 1. 黎曼积分的定义
#### 分割与黎曼和
设f: [a,b] → ℝ，P = {x₀, x₁, ..., xₙ}是[a,b]的一个分割，Δxᵢ = xᵢ - x_{i-1}，选择点ξᵢ ∈ [x_{i-1}, xᵢ]，定义黎曼和：
S(P,f) = ∑_{i=1}ⁿ f(ξᵢ)Δxᵢ

**课件原证明语句**：
"黎曼和依赖于分割P和选点ξ，积分定义为当‖P‖→0时黎曼和的极限"

#### 黎曼积分
若存在I ∈ ℝ，使得对于任意ε>0，存在δ>0，使得对于任何模‖P‖<δ的分割P，有|S(P,f) - I| < ε，则称f在[a,b]上黎曼可积，记作∫_a^b f(x)dx = I。

**课件原证明语句**：
"积分存在意味着所有黎曼和在分割足够细时都接近同一个数I"

### 2. 可积性条件
#### 定理（可积的充要条件）
f在[a,b]上可积 ⇔ ∀ε>0, ∃分割P使得U(P,f) - L(P,f) < ε，其中：
- U(P,f) = ∑ MᵢΔxᵢ（上积分和）
- L(P,f) = ∑ mᵢΔxᵢ（下积分和）
- Mᵢ = sup{f(x) | x ∈ [x_{i-1}, xᵢ]}
- mᵢ = inf{f(x) | x ∈ [x_{i-1}, xᵢ]}

**课件原证明语句**：
"上积分∫̅f = inf{U(P,f)}，下积分∫f = sup{L(P,f)}，可积⇔∫̅f=∫f"

#### 推论
1. 连续函数在闭区间上可积
2. 单调函数在闭区间上可积
3. 有界且只有有限个间断点的函数可积

**课件原证明语句**：
"连续函数在闭区间上一致连续，故振幅可任意小；单调函数的振幅和不超过|f(b)-f(a)|·‖P‖"

### 3. 平方根函数的连续性
#### 定理
平方根函数√x在[0,∞)上连续。

**课件原证明语句**：
"对x>0，|√x-√y| = |x-y|/(√x+√y) ≤ |x-y|/√x，当y→x时→0"

**证明思路**：
- 在x>0处，利用导数存在性证明连续性
- 在x=0处，直接验证极限定义

### 4. 曲线长度
#### 定义
设γ: [a,b] → ℝⁿ是连续可微曲线，其长度定义为：
L(γ) = ∫_a^b ‖γ'(t)‖ dt

其中‖γ'(t)‖是速度向量的欧几里得范数。

**课件原证明语句**：
"曲线长度定义为所有折线逼近的上确界：L(γ)=sup{∑‖γ(t_i)-γ(t_{i-1})‖ : a=t₀<...<tₙ=b}"

#### 性质
- 长度与参数化无关
- 直线段的长度等于两点间距离
- 连续可微曲线的长度有限

**课件原证明语句**：
"参数不变性：若φ:[α,β]→[a,b]是C¹微分同胚，则L(γ∘φ)=L(γ)"

## 重要概念总结
1. **黎曼积分**：通过黎曼和极限定义的积分
2. **达布上下和**：判断可积性的重要工具
3. **可积函数类**：连续函数、单调函数等
4. **曲线长度**：通过积分定义的几何量
