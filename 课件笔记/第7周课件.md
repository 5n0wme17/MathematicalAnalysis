# 第7周课件 - 单调函数、极限与导数

## 单词表
| 中文术语 | 英文术语 | 定义/说明 |
|---------|---------|---------|
| 单调函数 | Monotone function | x₁ < x₂ ⇒ f(x₁) ≤ f(x₂)（递增）或 f(x₁) ≥ f(x₂)（递减） |
| 严格单调 | Strictly monotone | x₁ < x₂ ⇒ f(x₁) < f(x₂)（严格递增）或 f(x₁) > f(x₂)（严格递减） |
| 左极限 | Left-hand limit | lim_{x→x₀⁻} f(x) = f(x₀⁻) |
| 右极限 | Right-hand limit | lim_{x→x₀⁺} f(x) = f(x₀⁺) |
| 导数 | Derivative | f'(x) = lim_{h→0} [f(x+h)-f(x)]/h |
| 可导 | Differentiable | 导数存在且有限 |
| 链式法则 | Chain rule | (f∘g)'(x) = f'(g(x))·g'(x) |
| 费马引理 | Fermat's lemma | 可导函数在极值点处导数为0 |
| 罗尔定理 | Rolle's theorem | f在[a,b]连续，(a,b)可导，f(a)=f(b) ⇒ ∃c∈(a,b), f'(c)=0 |
| 极值点 | Extremum point | 函数取得局部最大值或最小值的点 |
| 跳跃间断 | Jump discontinuity | f(x⁺) ≠ f(x⁻)（左右极限存在但不相等） |
| 可去间断 | Removable discontinuity | f(x⁺) = f(x⁻) ≠ f(x)（左右极限存在且相等） |
| 第一类间断点 | Discontinuity of the first kind | 左右极限都存在的间断点 |
| 第二类间断点 | Discontinuity of the second kind | 至少一侧极限不存在的间断点 |
| 切线斜率 | Slope of tangent line | 导数f'(x)的几何意义 |
| 微分 | Differential | df(x)(h) = f'(x)·h，函数在x点的线性近似 |

## 课件概述
本课件主要讨论单调函数的性质、无穷极限、函数在无穷远处的极限，以及实函数导数的基本概念、性质和运算法则。

## 主要内容

### 1. 单调函数
#### 定义
设f: E ⊂ ℝ → ℝ，若对任意x₁, x₂ ∈ E，x₁ < x₂ ⇒ f(x₁) ≤ f(x₂)，则称f在E上单调递增；若f(x₁) ≥ f(x₂)，则称f单调递减。

**课件原证明语句**：
"单调函数的间断点只能是跳跃间断点：设x₀为间断点，则f(x₀⁻)≤f(x₀)≤f(x₀⁺)（递增情形）"

#### 性质
- 单调函数至多有可数个间断点
- 单调函数的间断点都是第一类间断点（跳跃间断）
- 单调函数在定义区间内几乎处处可导

**课件原证明语句**：
"利用Vitali覆盖引理证明单调函数几乎处处可导"

### 2. 无穷极限与无穷远处的极限
#### 左极限的等价刻画
设f在(a,b)上有定义，x ∈ (a,b)，则以下等价：
f(x⁻) = A ⇔ ∀ε>0, ∃δ>0 (a < x-δ)，使得当x-δ < t < x时，有|f(t)-A| < ε

**课件原证明语句**：
"序列语言：f(x⁻)=A ⇔ 对任意tₙ→x⁻(tₙ<x)，有f(tₙ)→A"

#### 证明
⇐：设tₙ ∈ (a,x), tₙ → x。对任意ε>0，存在δ>0使得x-δ < t < x ⇒ |f(t)-A|<ε。
由于tₙ → x，存在N使得n≥N时x-δ < tₙ < x，故|f(tₙ)-A|<ε。

⇒：反证法。若结论不成立，则存在ε>0，对任意δ>0，存在t ∈ (x-δ,x)使|f(t)-A|≥ε。
取δₙ = 1/n，可得tₙ ∈ (x-1/n,x)使|f(tₙ)-A|≥ε，与f(x⁻)=A矛盾。

#### 类似地
右极限f(x⁺)有类似刻画。

**课件原证明语句**：
"右极限的刻画只需将x-δ < t < x改为x < t < x+δ"

### 3. 导数定义
#### 定义
设f在点x的某邻域内有定义，若极限
f'(x) = lim_{h→0} [f(x+h)-f(x)]/h
存在（有限），则称f在x处可导，f'(x)称为f在x处的导数。

**课件原证明语句**：
"可导⇒连续：f(x+h)-f(x)=[f(x+h)-f(x)]/h·h → f'(x)·0=0"

#### 几何意义
导数表示函数图像在点(x,f(x))处切线的斜率。

**课件原证明语句**：
"切线方程：y-f(x)=f'(x)(t-x)"

### 4. 导数的运算法则
#### 定理
设f,g在x处可导，则：
1. (f±g)'(x) = f'(x) ± g'(x)
2. (fg)'(x) = f'(x)g(x) + f(x)g'(x)
3. 若g(x)≠0，则(f/g)'(x) = [f'(x)g(x) - f(x)g'(x)]/[g(x)]²

**课件原证明语句**：
"乘积法则证明：f(x+h)g(x+h)-f(x)g(x)=[f(x+h)-f(x)]g(x)+f(x)[g(x+h)-g(x)]+o(h)"

#### 链式法则
若f在g(x)处可导，g在x处可导，则复合函数h=f∘g在x处可导，且
h'(x) = f'(g(x))·g'(x)

**课件原证明语句**：
"链式法则证明：设u=g(x)，则[f(g(x+h))-f(g(x))]/h = [f(u+k)-f(u)]/k · k/h，其中k=g(x+h)-g(x)"

### 5. 极值定理
#### 定理（费马引理）
设f在(a,b)上可导，且在c∈(a,b)处取得局部极值，则f'(c)=0。

**课件原证明语句**：
"若f'(c)>0，则存在δ>0使在(c-δ,c)上f(x)<f(c)，在(c,c+δ)上f(x)>f(c)，与c为极值点矛盾"

**注**：对局部最小值也有类似结论。

#### 反例说明
定理的逆命题不成立：f'(c)=0不一定意味着c是极值点（如f(x)=x³在x=0处）。

**课件原证明语句**：
"f(x)=x³在x=0处f'(0)=0，但x=0不是极值点（拐点）"

### 6. 罗尔定理
#### 定理
设f在[a,b]上连续，在(a,b)内可导，且f(a)=f(b)，则存在c∈(a,b)使得f'(c)=0。

**课件原证明语句**：
"由最值定理，f在[a,b]上取得最大值M和最小值m。若M=m则f为常数；否则M>m，由于f(a)=f(b)，极值点必在(a,b)内"

#### 推论（反证法应用）
若f在[a,b]上可导且f'(x)≠0对一切x∈(a,b)成立，则f在[a,b]上严格单调。

**课件原证明语句**：
"若f不严格单调，则存在x₁<x₂<x₃使f(x₁)≤f(x₂)≥f(x₃)或f(x₁)≥f(x₂)≤f(x₃)，由罗尔定理得矛盾"

## 重要概念总结
1. **单调函数**：保持序关系的函数，具有较好的正则性
2. **左右极限**：用ε-δ语言和序列语言等价刻画
3. **导数**：函数变化率的极限，几何意义为切线斜率
4. **导数的运算法则**：和差积商的求导公式
5. **极值定理**：可导函数在极值点处导数为零
6. **罗尔定理**：导函数零点存在性定理
