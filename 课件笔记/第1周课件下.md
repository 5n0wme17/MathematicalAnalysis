## 课件概述
本课件继续讨论集合论中的基数概念，并引入度量空间的基本定义和性质，包括开集、闭集、紧集等概念。

## 主要内容

### 1. 基数理论深入
#### 定义
- **基数相等**：X ∼ Y ⇔ 存在X到Y的双射
- **有限集**：基数等于某个自然数n的集合
- **可数集**：与自然数集N基数相同的无限集
- **不可数集**：无限但不可数的集合

#### 重要定理
1. **实数集不可数**：[0,1]区间不可数 ⇒ R不可数
2. **可数集的可数并仍可数**：若{E_k}为可数集族，则∪E_k可数
3. **可数集的有限笛卡尔积可数**：若A可数，则Aⁿ可数

#### 例子
- 有理数集Q可数
- 无理数集不可数（基数为c）
- 二进制序列集合{0,1}^N不可数

### 2. 度量空间基础
#### 定义
**度量空间**：(X,d)，其中d: X×X → [0,∞)满足：
1. 正定性：d(x,y)=0 ⇔ x=y
2. 对称性：d(x,y)=d(y,x)
3. 三角不等式：d(x,z) ≤ d(x,y) + d(y,z)

#### 例子
1. **欧氏空间**：Rⁿ，d(x,y) = (∑|x_i-y_i|²)^(1/2)
2. **离散度量空间**：d(x,y) = 1 if x≠y, 0 if x=y
3. **连续函数空间**：C[a,b]上的sup度量d(f,g)=max|f-g|
4. **ℓp空间**：d_p(x,y) = (∑|x_i-y_i|^p)^(1/p)

### 3. 开集与闭集
#### 定义
- **开集**：∀x∈U, ∃ε>0使B(x,ε)⊂U
- **闭集**：补集为开集
- **闭包**：E̅ = E ∪ E'（E'为极限点集）
- **内部**：E° = E的最大开子集
- **边界**：∂E = E̅ \ E°

#### 性质
1. 任意开集的并仍开；有限开集的交仍开
2. 任意闭集的交仍闭；有限闭集的并仍闭
3. E闭 ⇔ E'⊂E
4. E闭 ⇔ E=E̅

### 4. 紧集
#### 定义
- **紧集**：任意开覆盖有有限子覆盖
- **相对紧性**：K⊂Y⊂X，K在X中紧 ⇔ K在Y中紧

#### 重要定理
1. **紧集必闭**：K紧 ⇒ K闭
2. **闭集的紧子集仍紧**：F闭，K紧 ⇒ F∩K紧
3. **有限交性质**：若紧集族中任意有限交非空，则整体交非空
4. **Bolzano-Weierstrass**：紧集的无限子集必有极限点

#### 例子
- Rⁿ中的有界闭集是紧集
- C[a,b]中的一致有界等度连续函数集是紧(Arzela-Ascoli)

## 应用说明
1. 基数理论为分析无限集合提供了工具
2. 度量空间概念是泛函分析的基础
3. 紧性在证明存在性定理中起关键作用
4. 开集、闭集的概念是拓扑学的基础
