## 课件概述
本课件主要介绍数学分析中的集合论基础，包括集合的基本概念、运算、映射、笛卡尔积和基数理论。

## 主要内容

### 1. 基本概念
- **集合定义**：任何对象的集合
- **表示方法**：
  - 列举法：{a, b, c}
  - 描述法：{x ∈ R : x² = 1}
- **特殊符号**：
  - ∈（属于）、∉（不属于）
  - ∀（对所有）、∃（存在）
- **特殊集合**：
  - 空集 ∅
  - 单点集 {1}
  - 全集 Ω

### 2. 子集
- **定义**：A ⊆ B 表示 ∀x ∈ A ⇒ x ∈ B
- **相等集合**：A = B 当且仅当 A ⊆ B 且 B ⊆ A
- **真子集**：A ⊂ B 且 A ≠ B

### 3. 幂集
- **定义**：P(X) = {A : A ⊆ X}
- **性质**：
  - ∅ ∈ P(X), X ∈ P(X)
  - x ∈ X ⇔ {x} ∈ P(X)
  - Y ⊆ X ⇔ Y ∈ P(X)

### 4. 集合运算
#### 并集 (∪)
- A ∪ B = {x : x ∈ A 或 x ∈ B}
- **运算律**：
  - 交换律：A ∪ B = B ∪ A
  - 结合律：(A ∪ B) ∪ C = A ∪ (B ∪ C)
  - 吸收律：A ∪ A = A
  - A ∪ ∅ = A, A ∪ Ω = Ω

#### 交集 (∩)
- A ∩ B = {x : x ∈ A 且 x ∈ B}
- **运算律**：
  - 交换律：A ∩ B = B ∩ A
  - 结合律：(A ∩ B) ∩ C = A ∩ (B ∩ C)
  - 吸收律：A ∩ A = A
  - A ∩ ∅ = ∅, A ∩ Ω = A

#### 差集 (\) 和补集 (ᶜ)
- A \ B = {x : x ∈ A 且 x ∉ B}
- Aᶜ = Ω \ A = {x : x ∉ A}
- **德摩根定律**：
  - (A ∪ B)ᶜ = Aᶜ ∩ Bᶜ
  - (A ∩ B)ᶜ = Aᶜ ∪ Bᶜ

### 5. 映射
- **定义**：f: X → Y 是从X到Y的规则，为每个x ∈ X指定唯一的y ∈ Y
- **特殊映射**：
  - 单射（injective）：f(x) = f(y) ⇒ x = y
  - 满射（surjective）：im(f) = Y
  - 双射（bijective）：既是单射又是满射
- **复合映射**：g ∘ f: X → V, x ↦ g(f(x))
- **逆映射**：f是双射 ⇔ 存在唯一的g: Y → X使得g ∘ f = I_X且f ∘ g = I_Y

### 6. 笛卡尔积
- **定义**：A × B = {(a, b) : a ∈ A, b ∈ B}
- **n元组**：A₁ × A₂ × ... × Aₙ = {(a₁, a₂, ..., aₙ) : aᵢ ∈ Aᵢ}
- **幂**：Aⁿ = A × A × ... × A（n次）

### 7. 基数
- **定义**：两个集合有相同基数当且仅当存在它们之间的双射
- **记号**：Card(A) 或 |A|
- **比较**：Card(A) ≤ Card(B) 当存在从A到B的单射

#### 可数集
- **定义**：与自然数集N有相同基数的集合
- **性质**：
  1. 可数集 ⇔ 元素可排成序列
  2. 可数集的有限并仍可数
  3. 可数集的可数并仍可数
  4. 任何无限集都包含可数子集（ℵ₀是最小的无限基数）
  5. 可数集的有限笛卡尔积仍可数

#### 连续统基数 c
- **定义**：区间[0,1]的基数
- **定理**：[0,1]不可数 ⇒ c > ℵ₀
- **具有基数c的集合**：[0,1], (0,1), R, Rⁿ等
- **结论**：无理数集的基数为c

#### 连续统假设
- **问题**：是否存在基数κ使得ℵ₀ < κ < c？
- **历史**：康托猜想不存在这样的κ（连续统假设）
- **哥德尔（1939）**：CH不能从集合论公理中证伪
- **科恩（1963）**：CH不能从集合论公理中证明

## 重要定理
1. **康托-施罗德-伯恩斯坦定理**：如果Card(X) ≤ Card(Y)且Card(Y) ≤ Card(X)，则Card(X) = Card(Y)
2. **幂集定理**：对于任何集合X，有|X| < |P(X)|

## 应用说明
- 集合论是数学分析的基础，为后续的实数理论、函数极限等概念提供严格的语言
- 基数理论帮助理解无限集合的不同"大小"
- 连续统假设是集合论中的基本未解决问题
