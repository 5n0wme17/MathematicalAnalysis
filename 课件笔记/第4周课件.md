## 课件概述
本课件主要介绍扩展实数系统、上确界与下确界的概念、上极限与下极限的性质，以及非负项级数收敛的判别法。

## 主要内容

### 1. 扩展实数系统
#### 定义
扩展实数系统ℝ̄ = ℝ ∪ {+∞, -∞}，其中：
- +∞和-∞不同于任何实数
- 定义运算：-(+∞) = -∞，-(-∞) = +∞

#### 性质
- 有限数：x ∈ ℝ
- 无限数：x = +∞或x = -∞

### 2. 上确界与下确界
#### 定义
对于E ⊂ ℝ̄：
1. 若E ⊂ ℝ，sup(E)按实数定义
2. 若E包含+∞，则sup(E) = +∞
3. 若E不包含+∞但包含-∞，则sup(E) = sup(E\{-∞})

下确界定义：inf(E) = -sup(-E)，其中-E = {-x : x ∈ E}

#### 例子
- E = {-1, -2, -3, ...} ∪ {-∞} ⇒ sup(E) = -1, inf(E) = -∞
- {0.9, 0.99, 0.999, ...} ⇒ inf = 0.9, sup = 1
- ℕ = {1, 2, 3, ...} ⇒ inf = 1, sup = +∞

### 3. 上极限与下极限
#### 定理（上极限性质）
设A = limsup aₙ，则：
1. 存在子序列aₙₖ → A
2. 任何收敛子序列的极限不超过A

**证明思路**：
1. 构造子序列：对每个k，找到nₖ使得A-1/k < aₙₖ < A+1/k
2. 反证法：假设存在子序列收敛到A+ε，导出矛盾

### 4. 非负项级数收敛性
#### 定理
非负项级数∑aₙ收敛 ⇔ 其部分和序列有界

**证明思路**：
- 非负项级数的部分和序列单调递增
- 单调有界序列必收敛

### 5. 序列收敛与子序列
#### 定理
若α = limsup aₙ > 1，则存在子序列aₙₖ → +∞

**证明思路**：
- 取β > α，构造满足aₙₖ > β的子序列

## 重要概念总结
1. **扩展实数**：ℝ̄ = ℝ ∪ {±∞}，完善极限理论
2. **确界**：sup和inf在扩展实数中的统一定义
3. **上下极限**：描述序列的极限行为
4. **非负级数**：收敛性等价于部分和有界

## 应用说明
1. 扩展实数系统使我们可以严格讨论"趋于无穷"的概念
2. 上极限和下极限是分析序列振荡行为的重要工具
3. 非负项级数的判别法简化了许多收敛性证明
