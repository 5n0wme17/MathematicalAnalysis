## 课件概述
本课件主要讨论度量空间中序列的收敛性、柯西序列、完备性，以及紧空间和完备空间的性质。

## 主要内容

### 1. 序列收敛
#### 定义
设{xₙ}是度量空间X中的序列，x∈X，称{xₙ}收敛到x，如果：
∀ε>0, ∃N∈ℕ, 使得∀n≥N, 有d(xₙ,x) < ε

#### 性质
1. **极限唯一性**：若xₙ → p且xₙ → p'，则p = p'
2. **收敛序列有界**：任何收敛序列都是有界的
3. **闭集刻画**：A闭 ⇔ 对任意{xₙ}⊂A，若xₙ→x，则x∈A

### 2. 子序列
#### 定义
设{nₖ}是严格递增的正整数序列，则{x_{nₖ}}称为{xₙ}的子序列。

#### 性质
1. {xₙ}收敛到p ⇔ 每个子序列都收敛到p
2. 紧空间中的序列必有收敛子序列
3. ℝᵏ中的有界序列必有收敛子序列（波尔查诺-魏尔斯特拉斯）

### 3. 柯西序列与完备性
#### 定义
- **柯西序列**：∀ε>0, ∃N, ∀m,n≥N, d(xₘ,xₙ)<ε
- **完备度量空间**：每个柯西序列都收敛

#### 性质
1. 收敛序列 ⇒ 柯西序列
2. 紧度量空间中的柯西序列必收敛
3. 完备空间的闭子集仍完备
4. ℝⁿ是完备的（欧几里得空间）

### 4. 闭集套定理
#### 定理
设A₁ ⊃ A₂ ⊃ ... 是完备度量空间X中的非空闭集套，且diam(Aₙ) → 0，则∩_{n=1}∞ Aₙ是单点集。

#### 证明思路
1. 在每个Aₙ中取点aₙ
2. 证明{aₙ}是柯西序列
3. 由完备性知aₙ收敛到某点a
4. 证明a ∈ ∩Aₙ且交集中只有a

### 5. 直径
#### 定义
diam(E) = sup_{x,y∈E} d(x,y)

#### 性质
1. diam(E) = diam(E̅)（闭包的直径不变）
2. 若E有界，则diam(E) < ∞
3. diam(E) = 0 ⇔ E是单点集

## 重要定理总结
1. **收敛序列性质**：极限唯一、有界、闭集刻画
2. **子序列定理**：紧空间中序列有收敛子序列
3. **完备性等价条件**：柯西序列收敛
4. **闭集套定理**：直径趋于0的闭集套交为单点

## 应用说明
1. 完备性在不动点定理中的应用
2. 闭集套定理在存在性证明中的作用
3. 柯西准则判断序列收敛性
4. 子序列方法研究序列极限行为
