# 数学分析精讲 - 总单词表

## 集合论基础
| 中文术语 | 英文术语 | 定义/说明 |
|---------|---------|---------|
| 集合 | Set | 任何对象的集合 |
| 元素 | Element | 属于集合的成员，x ∈ A |
| 子集 | Subset | A ⊆ B 表示 ∀x ∈ A ⇒ x ∈ B |
| 并集 | Union | A ∪ B = {x : x ∈ A 或 x ∈ B} |
| 交集 | Intersection | A ∩ B = {x : x ∈ A 且 x ∈ B} |
| 补集 | Complement | Aᶜ = {x : x ∉ A} |
| 映射 | Map/Function | f: X → Y 的规则 |
| 笛卡尔积 | Cartesian Product | A × B = {(a,b) : a ∈ A, b ∈ B} |
| 基数 | Cardinal Number | 集合的"大小"，记为Card(A)或\|A\| |
| 空集 | Empty set | 不含任何元素的集合，记为∅ |
| 单点集 | Singleton | 只含一个元素的集合，如{1} |
| 全集 | Universal set | 考虑的所有对象的集合，记为Ω |
| 幂集 | Power set | 所有子集的集合，记为P(X)或2^X |
| 可数集 | Countable set | 与自然数集N有相同基数的集合 |
| 连续统基数 | Continuum | 区间[0,1]的基数，记为c |
| 阿列夫零 | ℵ₀ | 可数无限集的基数 |
| 双射 | Bijection | 既是单射又是满射的映射 |
| 单射 | Injection | f(x)=f(y) ⇒ x=y 的映射 |
| 满射 | Surjection | im(f) = Y 的映射 |
| 复合映射 | Composition | g ∘ f: X → V, x ↦ g(f(x)) |
| 逆映射 | Inverse map | f⁻¹: Y → X 满足 f⁻¹ ∘ f = I_X |
| 原像 | Pre-image | f⁻¹(C) = {x : f(x) ∈ C} |

## 度量空间与拓扑
| 中文术语 | 英文术语 | 定义/说明 |
|---------|---------|---------|
| 有限集 | Finite sets | 基数等于某个自然数n的集合 |
| 可数集 | Countable sets | 与自然数集N有相同基数的无限集 |
| 不可数集 | Uncountable sets | 无限但不可数的集合 |
| 基数相等 | Equal cardinality | X ∼ Y ⇔ 存在X到Y的双射 |
| 度量空间 | Metric space | 带有距离函数d: X×X → [0,∞)的集合X |
| 距离函数 | Distance function | 满足正定性、对称性、三角不等式的函数d |
| 离散度量 | Discrete metric | d(x,y)=1 if x≠y, 0 if x=y |
| 开集 | Open set | ∀x∈U, ∃ε>0使B(x,ε)⊂U |
| 闭集 | Closed set | 补集为开集的集合 |
| 开球 | Open ball | B(x,r) = {y∈X : d(x,y)<r} |
| 闭球 | Closed ball | {y∈X : d(x,y)≤r} |
| 凸集 | Convex set | λx+(1-λ)y∈E, ∀x,y∈E, 0<λ<1 |
| k-胞体 | k-cell | {x∈ℝᵏ : aᵢ≤xᵢ≤bᵢ, i=1,...,k} |
| 内点 | Interior point | ∃r>0使B(x,r)⊂E的点x |
| 外点 | Exterior point | ∃r>0使B(x,r)⊂Eᶜ的点x |
| 边界点 | Boundary point | 既不是内点也不是外点的点 |
| 闭包 | Closure | E̅ = E ∪ E'（E'为极限点集） |
| 稠密集 | Dense set | 闭包等于全空间的集合，E̅ = X |
| 无处稠密集 | Nowhere dense set | 闭包的内部为空的集合，(E̅)° = ∅ |
| 附着点 | Adherent point | 每个邻域都与E相交的点 |
| 聚点/极限点 | Accumulation point / Limit point | 每个邻域都包含E中异于该点的点 |
| 孤立点 | Isolated point | 存在邻域只包含该点自身的点 |
| 完备集 | Perfect set | 闭集且每个点都是极限点 |
| 开覆盖 | Open cover | 开集族{G_α}使得E ⊂ ∪_α G_α |
| 紧集 | Compact set | 任意开覆盖都有有限子覆盖 |
| 有限子覆盖 | Finite subcover | 开覆盖的有限子集仍覆盖原集合 |
| 相对开集 | Open relative to | E在Y中开 ⇔ E = Y∩G，G在X中开 |
| 上确界 | Supremum | 最小上界，记为sup E |
| 下确界 | Infimum | 最大下界，记为inf E |

## 函数与极限
| 中文术语 | 英文术语 | 定义/说明 |
|---------|---------|---------|
| 函数极限 | Limit of function | lim_{x→a} f(x) = L |
| 连续性 | Continuity | lim_{x→a} f(x) = f(a) |
| 一致连续 | Uniform continuity | ∀ε>0, ∃δ>0, ∀x,y, d(x,y)<δ ⇒ |f(x)-f(y)|<ε |
| 导数 | Derivative | f'(a) = lim_{h→0} [f(a+h)-f(a)]/h |
| 微分 | Differential | df = f'(x)dx |
| 中值定理 | Mean Value Theorem | f(b)-f(a) = f'(c)(b-a) |
| 泰勒展开 | Taylor expansion | f(x) = Σ_{k=0}^n [f^(k)(a)/k!](x-a)^k + R_n(x) |
| 黎曼积分 | Riemann integral | ∫_a^b f(x)dx = lim_{n→∞} Σ_{i=1}^n f(x_i^*)Δx_i |
| 可积性 | Integrability | 上积分=下积分 |
| 原函数 | Antiderivative | F'(x) = f(x) |
| 反常积分 | Improper integral | 积分区间无限或被积函数无界 |

## 级数与序列
| 中文术语 | 英文术语 | 定义/说明 |
|---------|---------|---------|
| 序列收敛 | Sequence convergence | lim_{n→∞} a_n = L |
| 子序列 | Subsequence | {a_{n_k}} where n_k → ∞ |
| 柯西序列 | Cauchy sequence | ∀ε>0, ∃N, ∀m,n>N, |a_m - a_n|<ε |
| 级数收敛 | Series convergence | Σ_{n=1}^∞ a_n = S if partial sums → S |
| 绝对收敛 | Absolute convergence | Σ|a_n| converges |
| 条件收敛 | Conditional convergence | Σa_n converges but Σ|a_n| diverges |
| 一致收敛 | Uniform convergence | ∀ε>0, ∃N, ∀n>N, ∀x, |f_n(x)-f(x)|<ε |
| 幂级数 | Power series | Σ_{n=0}^∞ c_n(x-a)^n |
| 收敛半径 | Radius of convergence | R = 1/lim sup |c_n|^{1/n} |

## 多元微积分
| 中文术语 | 英文术语 | 定义/说明 |
|---------|---------|---------|
| 偏导数 | Partial derivative | ∂f/∂x_i = lim_{h→0} [f(x+he_i)-f(x)]/h |
| 方向导数 | Directional derivative | D_v f(a) = lim_{h→0} [f(a+hv)-f(a)]/h |
| 梯度 | Gradient | ∇f = (∂f/∂x_1, ..., ∂f/∂x_n) |
| 雅可比矩阵 | Jacobian matrix | J_f = [∂f_i/∂x_j]_{i,j} |
| 隐函数定理 | Implicit Function Theorem | F(x,y)=0 ⇒ y=f(x) locally |
| 拉格朗日乘数 | Lagrange multiplier | ∇f = λ∇g for constrained extrema |
| 重积分 | Multiple integral | ∫...∫_D f(x_1,...,x_n) dx_1...dx_n |
| 曲线积分 | Line integral | ∫_C f ds or ∫_C F·dr |
| 曲面积分 | Surface integral | ∫∫_S f dS or ∫∫_S F·dS |
| 格林定理 | Green's Theorem | ∫_C Pdx+Qdy = ∫∫_D (∂Q/∂x - ∂P/∂y) dA |
| 斯托克斯定理 | Stokes' Theorem | ∫_C F·dr = ∫∫_S (∇×F)·dS |
| 高斯定理 | Gauss' Theorem | ∫∫∫_V (∇·F) dV = ∫∫_∂V F·dS |
